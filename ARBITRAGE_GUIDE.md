# ETF/LOF 套利操作完全指南

## 一、套利前准备

### 1.1 需要的账户

```
必备账户：
├── 证券账户（场内交易用）
│   ├── 任意券商即可
│   ├── 推荐低佣金券商（万1-万2.5）
│   └── 开通场内基金交易权限
│
└── 场外基金账户（场外申赎用）
    ├── 可用证券账户关联的场外账户
    ├── 或天天基金/支付宝等平台
    └── 注意：LOF需在场外单独开户
```

### 1.2 重要概念

```
T日：交易日（工作日 15:00 前）
T+1：下一个交易日
T+2：下两个交易日

场内：股票账户买卖（像买卖股票一样）
场外：基金账户申购/赎回（向基金公司买卖）

净值（NAV）：基金的真实价值，每日公布一次
IOPV：实时参考净值，盘中实时估算
市价：场内买卖的实时成交价
```

### 1.3 费用结构

```
场内交易费用：
├── 佣金：万1 - 万2.5（券商不同）
├── 最低：5元/笔
└── 无印花税、无过户费

场外申购费用：
├── 申购费：0.6% - 1.5%（平台不同）
├── 常见折扣：1折（0.06% - 0.15%）
└── 大额申购可能优惠

场外赎回费用：
├── 赎回费：0 - 1.5%（持有时间不同）
├── 持有 > 7天：通常 0.5% 或更低
├── 持有 > 30天：通常 0% 或很低
└── 注意：部分基金有惩罚性赎回费
```

## 二、溢价套利（最常见）

### 2.1 什么是溢价套利

```
溢价 = 场内价格 > 基金净值

场景：
┌─────────────────────────────────────┐
│  黄金LOF (164701)                   │
│                                     │
│  场内价格: ¥2.80  ←────────────┐    │
│  基金净值: ¥2.50  ←─────┐      │    │
│                         │      │    │
│  溢价率: (2.80-2.50)/2.50 = 12% │    │
│                         ↓      ↓    │
│  操作: 场外申购(便宜) → 场内卖出(贵)│
└─────────────────────────────────────┘

利润 = 卖出价 - 买入价 - 手续费
```

### 2.2 完整操作流程

```
═══════════════════════════════════════════════════════
溢价套利操作流程
═══════════════════════════════════════════════════════

第1步：发现机会（套利监控系统）
─────────────────────────────────────────────────────
时间：T日 上午 10:00
操作：监控系统发现某LOF溢价率达到 8%

  ┌──────────────────────────────────────┐
  │  原油LOF (161129)                    │
  │  场内价格: ¥0.65                     │
  │  基金净值: ¥0.60（昨日公布）          │
  │  预估净值: ¥0.60（盘中估算）          │
  │  溢价率: 8.3%                        │
  │  套利空间: 约 6-7%（扣除费用后）       │
  └──────────────────────────────────────┘

第2步：计算收益
─────────────────────────────────────────────────────
投入金额：10万元

收入：
├── 场内卖出：100,000 ÷ 0.60 × 0.65 = ¥108,333
│   （申购10万份，按0.65卖出）
└── 毛利润：¥8,333

费用：
├── 场外申购费：¥100,000 × 0.1% = ¥100（1折）
├── 场内卖出佣金：¥108,333 × 0.025% = ¥27
├── 场内卖出最低佣金：¥5（已满足）
└── 总费用：¥127

净利润：¥8,333 - ¥127 = ¥8,206
收益率：8.2%
年化收益：8.2% × 365 ÷ 3 ≈ 998%

第3步：场外申购（T日 15:00前）
─────────────────────────────────────────────────────
平台选择：
├── 方式A：证券APP → 场外基金 → 申购
├── 方式B：天天基金/支付宝 → 搜索基金代码 → 申购
└── 注意：必须用同一个身份的账户

操作步骤（以证券APP为例）：
┌──────────────────────────────────────┐
│  1. 打开证券APP                       │
│  2. 搜索 "161129"                     │
│  3. 选择 "场外申购"（不是场内买入！）  │
│  4. 输入金额：100,000元               │
│  5. 确认申购                          │
│                                      │
│  ⚠️ 重要：15:00 前提交，按当日净值计算 │
└──────────────────────────────────────┘

第4步：等待份额到账（T+2日）
─────────────────────────────────────────────────────
时间线：
├── T日：提交申购申请
│   └── 按 T日 净值确认份额
├── T+1日：基金公司确认份额
│   └── 晚上可在账户看到份额
└── T+2日：份额到账，可操作
    └── 早上即可场内卖出

  T日          T+1日         T+2日
   │             │            │
   ▼             ▼            ▼
┌─────┐      ┌─────┐      ┌─────┐
│申购 │ ──→  │确认 │ ──→  │到账 │
│提交 │      │份额 │      │可卖 │
└─────┘      └─────┘      └─────┘

第5步：转托管（如需要）
─────────────────────────────────────────────────────
如果场外和场内账户不互通，需要转托管：

┌──────────────────────────────────────┐
│  转托管操作：                         │
│                                      │
│  1. 证券APP → 场外基金持仓            │
│  2. 选择基金 → 转托管                │
│  3. 输入转出数量                      │
│  4. 选择转入席位（券商席位号）        │
│  5. 提交申请                         │
│                                      │
│  ⚠️ 转托管需要 T+2 日到账            │
│  ⚠️ 部分券商可直接跨市场卖出          │
└──────────────────────────────────────┘

第6步：场内卖出（T+2日 开盘后）
─────────────────────────────────────────────────────
操作：
┌──────────────────────────────────────┐
│  1. 证券APP → 持仓                    │
│  2. 选择该 LOF 基金                   │
│  3. 点击 "卖出"                       │
│  4. 输入数量、价格                    │
│  5. 确认卖出                          │
│                                      │
│  💡 卖出技巧：                        │
│  ├── 开盘集合竞价卖出（9:15-9:25）    │
│  ├── 或开盘后立即市价卖出             │
│  └── 避免长时间挂单，价格可能回落     │
└──────────────────────────────────────┘

第7步：结算
─────────────────────────────────────────────────────
卖出后资金到账：
├── A股：T+1 可用，T+2 可取
└── 实际到账时间以券商为准

完整周期：T + 2 ~ 3 日
═══════════════════════════════════════════════════════
```

### 2.3 关键注意事项

```
⚠️ 风险1：净值变动风险
─────────────────────────────
T日申购后，T日净值还没公布
如果当晚公布的净值比预期高，利润会缩水

应对：
├── 选择跟踪误差小的指数基金
├── 避开波动大的主动基金
└── 留出足够的安全边际（溢价率 > 3%）

⚠️ 风险2：价格回落风险
─────────────────────────────
T+2 卖出时，场内价格可能已回落
原来溢价 8%，现在可能只剩 2%

应对：
├── 选择溢价持续时间长的基金
├── 查看历史溢价走势
├── 开盘立即卖出，不要等待
└── 设置止损价格

⚠️ 风险3：流动性风险
─────────────────────────────
场内成交量太小，卖不出去
或卖出时价格剧烈波动

应对：
├── 选择日成交量 > 1000万 的基金
├── 分批卖出，避免砸盘
└── 避开成交量极小的基金

⚠️ 风险4：暂停申购风险
─────────────────────────────
基金公司可能暂停大额申购
或完全暂停申购

应对：
├── 申购前查看基金公告
├── 关注限额（常见限额 1-10万）
└── 准备备选基金
```

## 三、折价套利

### 3.1 什么是折价套利

```
折价 = 场内价格 < 基金净值

场景：
┌─────────────────────────────────────┐
│  某只 LOF 基金                       │
│                                     │
│  场内价格: ¥0.90  ←────────────┐    │
│  基金净值: ¥1.00  ←─────┐      │    │
│                         │      │    │
│  折价率: (1.00-0.90)/1.00 = 10% │    │
│                         ↓      ↓    │
│  操作: 场内买入(便宜) → 场外赎回(贵)│
└─────────────────────────────────────┘

利润 = 赎回金额 - 买入金额 - 手续费
```

### 3.2 完整操作流程

```
═══════════════════════════════════════════════════════
折价套利操作流程
═══════════════════════════════════════════════════════

第1步：发现机会
─────────────────────────────────────────────────────
  ┌──────────────────────────────────────┐
  │  某只 LOF 基金                        │
  │  场内价格: ¥0.90                     │
  │  基金净值: ¥1.00                     │
  │  折价率: 10%                         │
  │  套利空间: 约 7-8%（扣除费用后）      │
  └──────────────────────────────────────┘

第2步：计算收益
─────────────────────────────────────────────────────
买入金额：10万元（场内买入）

成本：
├── 场内买入：100,000元
├── 买入佣金：100,000 × 0.025% = 25元
├── 最低佣金：5元（已满足）
└── 总成本：100,025元

收入：
├── 买入份额：100,000 ÷ 0.90 = 111,111份
├── 赎回金额：111,111 × 1.00 = 111,111元
├── 赎回费：111,111 × 0.5% = 556元（持有<7天）
│   或 0元（持有>30天）
└── 净收入：110,555元（短期）或 111,111元（长期）

净利润（短期）：110,555 - 100,025 = 10,530元
收益率：10.5%

⚠️ 注意：短期赎回费高，建议持有 > 7天

第3步：场内买入（T日）
─────────────────────────────────────────────────────
操作：
┌──────────────────────────────────────┐
│  1. 证券APP → 交易                    │
│  2. 搜索基金代码                      │
│  3. 选择 "买入"                       │
│  4. 输入数量（必须是100的整数倍）      │
│  5. 输入价格或市价买入                │
│  6. 确认买入                          │
│                                      │
│  💡 技巧：                            │
│  ├── 盘中分批买入，降低成本           │
│  └── 注意成交量，避免买入困难         │
└──────────────────────────────────────┘

第4步：转托管到场外（T日 或 T+1日）
─────────────────────────────────────────────────────
操作：
┌──────────────────────────────────────┐
│  1. 证券APP → 持仓                    │
│  2. 选择该 LOF 基金                   │
│  3. 点击 "转托管"                     │
│  4. 选择 "转出场内"                   │
│  5. 输入数量                          │
│  6. 提交申请                          │
│                                      │
│  ⚠️ 转托管 T+2 到账                  │
└──────────────────────────────────────┘

第5步：场外赎回（T+3日 或之后）
─────────────────────────────────────────────────────
等待份额到场外账户后：

┌──────────────────────────────────────┐
│  1. 证券APP → 场外基金持仓            │
│  2. 选择该基金                        │
│  3. 点击 "赎回"                       │
│  4. 输入份额                          │
│  5. 确认赎回                          │
│                                      │
│  ⚠️ 赎回按 T日 净值计算              │
│  ⚠️ 资金 T+1 ~ T+3 到账              │
└──────────────────────────────────────┘

完整周期：T + 3 ~ 5 日
═══════════════════════════════════════════════════════
```

### 3.3 折价套利 vs 溢价套利

```
┌────────────────┬─────────────────┬─────────────────┐
│                │    溢价套利      │    折价套利      │
├────────────────┼─────────────────┼─────────────────┤
│ 操作方向       │ 申购 → 卖出      │ 买入 → 赎回      │
├────────────────┼─────────────────┼─────────────────┤
│ 资金占用时间   │ 2-3 天          │ 5-7 天          │
├────────────────┼─────────────────┼─────────────────┤
│ 操作复杂度     │ 简单            │ 较复杂（需转托管）│
├────────────────┼─────────────────┼─────────────────┤
│ 赎回费影响     │ 无              │ 有（持有<7天高） │
├────────────────┼─────────────────┼─────────────────┤
│ 主要风险       │ 价格回落        │ 净值下跌         │
├────────────────┼─────────────────┼─────────────────┤
│ 适合人群       │ 新手            │ 有经验者         │
├────────────────┼─────────────────┼─────────────────┤
│ 机会频率       │ 较多            │ 较少            │
└────────────────┴─────────────────┴─────────────────┘

结论：新手优先选择溢价套利
```

## 五、高级技巧

### 5.1 如何选择套利标的

```
优选标准：
┌─────────────────────────────────────────────────┐
│  ✅ 溢价/折价率 > 3%（扣除费用后有利润空间）     │
│  ✅ 日成交额 > 1000万（流动性好）               │
│  ✅ 跟踪误差 < 0.5%（指数基金）                  │
│  ✅ 历史溢价持续时间 > 3天                      │
│  ✅ 无暂停申购公告                              │
│  ✅ 非QDII基金（避免汇率风险）                  │
└─────────────────────────────────────────────────┘

避雷指南：
┌─────────────────────────────────────────────────┐
│  ❌ 溢价率 > 20%（可能有猫腻，风险极高）         │
│  ❌ 日成交额 < 100万（卖不出去）                │
│  ❌ 小规模基金（< 1亿，易被操纵）               │
│  ❌ 刚上市的新基金（价格波动大）                │
│  ❌ 有重大公告的基金（可能停牌）                │
└─────────────────────────────────────────────────┘
```

### 5.2 时机选择

```
最佳套利时机：

1. 开盘初期（9:30-10:00）
   ├── 价格波动大，溢价/折价率最高
   ├── 适合发现机会
   └── 但需快速决策

2. 尾盘时段（14:30-15:00）
   ├── 价格趋于稳定
   ├── 溢价/折价率相对确定
   └── 申购截止前最后机会

3. 重大事件后
   ├── 市场大跌后（折价套利机会）
   ├── 板块大涨后（溢价套利机会）
   └── QDII 美股大涨后（次日溢价）

避免时段：
├── 市场剧烈波动时（风险大）
├── 节假日前夕（资金占用时间长）
└── 基金分红前后（价格异常）
```

### 5.3 资金管理

```
单次套利资金建议：
├── 新手：1-5万（试水）
├── 进阶：5-20万
├── 专业：20-100万
└── 注意申购限额

资金分配策略：
├── 单只基金不超过总资金 20%
├── 同时操作 3-5 只基金分散风险
├── 保留 20% 现金应对机会
└── 资金滚动操作（提高周转率）

收益预期：
├── 保守：月收益 2-5%
├── 中等：月收益 5-10%
└── 激进：月收益 10-20%
```

### 5.4 风险控制

```
止损策略：
┌─────────────────────────────────────────────────┐
│  溢价套利：                                     │
│  ├── T+2 开盘时溢价率 < 2%，立即卖出           │
│  ├── 接受小幅亏损，避免更大损失                │
│  └── 不要抱有侥幸心理                          │
│                                                 │
│  折价套利：                                     │
│  ├── 净值跌破买入价 3%，考虑止损               │
│  ├── 设置心理止损线                            │
│  └── 分批赎回，降低风险                        │
└─────────────────────────────────────────────────┘

风险对冲：
├── 同时做溢价和折价套利（对冲市场风险）
├── 套利 + 短期债券（降低资金成本）
└── 保持冷静，不追涨杀跌
```

## 六、常见问题

### Q1：套利需要多少钱起步？

```
最低门槛：
├── 场外申购：10元起（部分平台 1元起）
├── 场内买入：100份起（约 100-200元）
└── 建议起步：1万元

实际操作建议：
├── 1-5万：学习阶段
├── 5-20万：正常操作
└── 20万+：注意申购限额
```

### Q2：套利有风险吗？

```
风险等级：中低风险

主要风险：
├── 净值波动风险（持有期间基金涨跌）
├── 价格回归风险（溢价/折价消失）
├── 流动性风险（卖不出去）
└── 操作风险（输错代码、错过时间）

风险控制：
├── 选择溢价/折价率 > 3% 的标的
├── 快速操作，缩短持有时间
├── 分散投资，不押注单一基金
└── 设置止损，及时止损
```

### Q3：每天都能套利吗？

```
套利机会频率：
├── 每天都有一定机会
├── 大机会（> 5%）每周 2-5 次
├── 小机会（2-5%）每天都有
└── 极端机会（> 10%）每月 1-3 次

建议：
├── 不必每天操作
├── 等待高概率机会
├── 宁可错过，不可做错
└── 积累经验后可提高频率
```

### Q4：需要盯盘吗？

```
传统方式：需要盯盘
├── 手动查数据
├── 手动计算
├── 手动操作
└── 时间成本高

使用套利监控系统：
├── 自动监控
├── 微信推送提醒
├── 快速决策
└── 不用盯盘

这也是套利监控系统的价值所在！
```
